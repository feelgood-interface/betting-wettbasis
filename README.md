# betting-wettbasis
Get best bets from [wettbasis.com](https://www.wettbasis.com/).

## Getting Started
1. Install the requirements
```
$ pip install -r requirements.txt
```
2. Run the script
```
$ python main.py
```

### Disclaimer
The script might not work with future website changes and was only developed for fun. Please keep fair use policy in mind.
